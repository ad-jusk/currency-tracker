## About
This project displays current and historical exchange rates of foreign currencies based on data fetched from Polish National Bank API. Data is fetched every day, historical values include data from up to 1 month. 

Click <a href="https://ad-jusk.github.io/currency-tracker/" target="blank">here</a> to try it out.

## Tech stack
- React 19
- Material UI
- Redux
- Firebase
- Github Actions (synchronise via cron, deploy)
